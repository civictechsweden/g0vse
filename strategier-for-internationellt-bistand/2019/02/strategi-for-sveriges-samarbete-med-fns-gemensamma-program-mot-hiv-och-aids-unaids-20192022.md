# Strategi för Sveriges samarbete med FN:s gemensamma program mot hiv och aids (UNAIDS) 2019–2022

Denna strategi ska ligga till grund för svenskt samarbete med FN:s gemensamma program mot hiv och aids (UNAIDS) för perioden 2019–2022\. Strategin fastställer prioriteringar och samarbetsformer för Sveriges partnerskap med UNAIDS på global, regional och nationell nivå.
