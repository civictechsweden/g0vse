# Sakråd inför dialog med FN:s rasdiskrimineringskommitté (CERD) och om regeringens handlingsplan mot rasism och hatbrottDiarienummer: A2025/00489, A2025/00489

Den 22 maj bjöd Regeringskansliet in till ett sakråd inför Sveriges dialog med CERD och om arbetet inom handlingsplanen mot rasism och hatbrott. Vid mötet gavs information om processen kring granskningen av Sveriges efterlevnad av rasdiskrimineringskonventionen och om den nya handlingsplanen mot rasism och hatbrott. Sakrådet var också ett tillfälle för civilsamhället att lämna synpunkter eller förslag och ställa frågor.
