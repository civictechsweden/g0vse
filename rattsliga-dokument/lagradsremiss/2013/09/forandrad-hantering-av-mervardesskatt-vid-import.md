# Förändrad hantering av mervärdesskatt vid import

I lagrådsremissen föreslås att Skatteverket ska vara beskattningsmyndighet för mervärdesskatt vid import när en importör är registrerad till mervärdesskatt i Sverige och agerar i egenskap av beskattningsbar person vid importen. Vidare föreslås ett informationsutbyte mellan Tullverket och Skatteverket för att myndigheterna ska kunna bedöma tillämplig beskattningsmyndighet och kontrollera redovisad mervärdesskatt vid import.

Ändringarna föreslås träda i kraft den 1 januari 2015\.
