# Tonnageskatt

Utredningen föreslår att Sverige inför ett system med tonnagebeskattning av fartyg. Liknande system finns i majoriteten av sjöfartsnationerna inom Europeiska unionen. Syftet med dessa system är att ge rederier i Europa bättre möjligheter att konkurrera med rederier i länder med s.k. bekvämlighetsflagg.
Tonnageskatt innebär att inkomsten från fartyg bestäms schablonmässigt med utgångspunkt från fartygens nettodräktighet. Rederierna betalar skatt bestämd på denna schabloninkomst i stället för på de faktiska inkomsterna från fartygen.
