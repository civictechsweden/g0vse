# Bättre konkurrens för ökat bostadsbyggande

En särskild utredare ska se över förutsättningarna för att främja konkurrensen på byggmarknaden och vid behov komma med förslag som kan förbättra situationen. Syftet är att förbättra förutsättningarna för att utbudet av nya bostäder långsiktigt ska anpassas till efterfrågan. Uppdraget ska redovisas senast den 1 oktober 2015\.
