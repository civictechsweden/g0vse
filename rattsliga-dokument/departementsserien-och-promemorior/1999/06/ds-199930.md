# Underhållsstöd vid växelvis boende - Underhållsstöd - till barn med särlevande föräldrar

Mot denna bakgrund har ett förslag till ändrade bestämmelser för hur underhållsstöd vid växelvis boende skall beräknas och fördelas mellan de båda föräldrarna utarbetats inom Socialdepartementet. Ett alternativt förslag innebärande att underhållsstöd vid växelvis boende slopas har även lämnats.
