# Konvention, utarbetad på grundval av artikel K 3.2 C i Fördraget om Europeiska unionen, om kamp mot korruption som tjänstemän i Europeiska gemenskaperna eller Europeiska unionens medlemsstater

Konvention, utarbetad på grundval av artikel K 3\.2 C i Fördraget om Europeiska unionen, om kamp mot korruption som tjänstemän i Europeiska gemenskaperna eller Europeiska unionens medlemsstater är delaktiga i.
