# Avtal med Jordanien om rättslig status för svenska polisinstruktörer som deltar i utbildningsinsatser för irakiska poliser ...

Avtal med Jordanien om rättslig status för svenska polisinstruktörer som deltar i utbildningsinsatser för irakiska poliser i Konungariket Jordanien.
