# Avtal med Förbundsrepubliken Tyskland för att undvika dubbelbeskattning beträffande skatter på inkomst och förmögenhet och på  arv och gåva samt för att lämna ömsesidig handräckning vid beskattning

Avtal med Förbundsrepubliken Tyskland för att undvika dubbelbeskattning beträffande skatter på inkomst och förmögenhet och på arv och gåva samt för att lämna ömsesidig handräckning vid beskattning. (Svenskt-tyskt skatteavtal).
Bonn den 14 juli 1992
