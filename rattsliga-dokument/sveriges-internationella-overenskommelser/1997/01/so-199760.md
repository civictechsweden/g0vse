# Anläggningsavtal med Organisationen för förbud mot kemiska vapen om inspektioner på platsen vid produktionsanläggningen för list

Anläggningsavtal med Organisationen för förbud mot kemiska vapen om inspektioner på platsen vid produktionsanläggningen för lista 1-ämnen för skyddsändamål vid försvarets forskningsantalts avdelning vid NBC-skydd i Umeå i Sverige.
Haag den 28 november 1997
