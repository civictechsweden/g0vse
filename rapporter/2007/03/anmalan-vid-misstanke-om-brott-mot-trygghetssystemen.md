# Anmälan vid misstanke om brott mot trygghetssystemen

Rätt person skall ha rätt ersättning vid rätt tillfälle. Det är utgångspunkten för det arbete Delegationen mot felaktiga utbetalningar bedriver. Denna rapport innehåller en vägledning för anmälan om misstänkta brott mot trygghetssystemen. Rapporten riktar sig till samtliga myndigheter och organisationer som beslutar om ersättningar från dessa system samt till polis- och åklagarmyndigheter.
