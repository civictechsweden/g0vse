# Bidrag till Världsbanken för World Development Report (WDR) 2012 om jämställdhet och utveckling

Regeringskansliet (Utrikesdepartementet) beslutar att lämna ett bidrag om 150 000 US\-dollar (högst 1 200 000 kronor) till Världsbanken för arbetet med World Development Report (WDR) 2012 med tema ’Gender Equality and Development’.
