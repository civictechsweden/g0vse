# Bidrag till Kristna Fredsrörelsen – stöd för NGO-representanter från ”the Global South” att närvara vid mötet med den förberedande kommittén för ett internationellt vapenhandelsavtal

Regeringskansliet (Utrikesdepartementet) beslutar att lämna ett bidrag på högst 200 000 kronor till Kristna Fredsrörelsen för att stödja deltagande av NGO-representanter från utvecklingsländer (”the Global South”) vid mötet med Förenta nationernas förberedande kommitté för ett internationellt vapenhandelsavtal (Arms Trade Treaty) i New York den 12 – 23 juli 2010.
