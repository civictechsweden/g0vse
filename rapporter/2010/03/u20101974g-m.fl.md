# Examensmål för gymnasieskolans nationella program

För gymnasieskolans nationella program ska de examensmål gälla som framgår av detta dokument.

Dokumentet är en bilaga till förordning om examensmål för gymnasieskolans nationella program. Dessa mål gäller utöver de gemensamma kunskapsmål för gymnasieskolans nationella program som anges i förordningen (SKOLFS 1994:2) om 1994 års läroplan för de frivilliga skolformerna.
