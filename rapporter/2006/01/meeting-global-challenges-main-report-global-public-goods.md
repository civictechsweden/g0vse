# Meeting Global Challenges Main Report Global Public Goods

Global Public Goods \- s k globala gemensamma nyttigheter innebär att de kan utnyttjas av alla. Nyttigheterna svarar mot många globala utmaningar, t ex skydd mot smittsamma sjukdomar, att tackla klimatförändringar, finansiell stabilitet etc.

The Task Force on Global Public Goods tillsattes 2003 och producerade ett antal expertrapporter om olika globala nyttigheter. Det här är deras huvudrapport som utkom 2006\.
