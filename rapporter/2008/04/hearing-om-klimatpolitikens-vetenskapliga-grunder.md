# Hearing om klimatpolitikens vetenskapliga grunder

Rapporten innehåller bidrag från experter som medverkade vid en hearing som Kommissionen för hållbar utveckling anordnade den 3 september 2007\. Vid hearingen diskuterades rapporten "Vetenskapligt underlag för klimatpolitiken" från regeringens vetenskapliga råd för klimatfrågor (Miljövårdsberedningens rapport 2007:03\); bidragen i föreliggande rapport ska läsas som kommentarer till detta underlag.

De analyser, synpunkter och förslag som framförs i rapporten ansvarar endast respektive författare själv för.
