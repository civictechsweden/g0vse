# Statssekreterare Anders Hall deltar i nordiskt ministermöte om migration

Regeringen deltar i det nordiska ministermötet för att stärka det nordiska samarbetet kring migrationsfrågor. Sverige lyfter flera prioriterade områden som är viktiga för att skapa en migrationspolitik som är långsiktigt hållbar och tydlig för både myndigheter och människor.

## Sveriges prioriteringar vid mötet:

### Genomförandet av EU:s migrationspakt

Sverige arbetar med att anpassa lagar och myndigheternas arbetssätt inför att EU:s nya regler börjar gälla. Det handlar bland annat om hur gränskontroller och mottagning av asylsökande ska fungera, och hur länder ska hjälpa varandra genom en solidaritetsmekanism.

### Återvändande

Regeringen har gett myndigheterna fler verktyg för att öka antalet personer som återvänder efter avslag på sin ansökan. Det handlar till exempel om att bygga fler förvar, skapa särskilda boenden för personer som ska återvända och att samarbeta med ursprungsländerna.

### Frivillig återvandring till Ukraina

Sverige stödjer EU:s rekommendation om att hjälpa personer från Ukraina som vill återvända när det är säkert. Det finns redan lagstiftning som gör det möjligt att få ett särskilt uppehållstillstånd om man deltar i ett återvändandeprogram.

### Samarbete med länder utanför EU

Sverige är positiv till nya sätt att samarbeta med andra länder för att minska irreguljär migration, alltså migration som sker utanför lagliga vägar. Ett exempel är att skapa informationscenter, så kallade enhetshubbar, som kan ge stöd till personer som funderar på att återvända.

### Nordiskt samarbete

Sverige lägger stort vikt vid nordiskt samarbete och vill stärka det nordiska samarbetet genom konkreta åtgärder för att hantera gemensamma utmaningar.

Ministermötet samlar migrationsministrar från Sverige, Danmark, Finland och Island, samt statssekreterare från Norge Finland är ordförande i NSHF under 2025, och Sverige tar över ordförandeskapet 2026.
