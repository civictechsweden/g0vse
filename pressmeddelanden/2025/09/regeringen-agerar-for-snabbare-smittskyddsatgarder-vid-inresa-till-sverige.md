# Regeringen agerar för snabbare smittskyddsåtgärder vid inresa till Sverige

– Vägledningen handlar om att vara förberedd. När någon kommer till Sverige och det finns misstanke om allvarlig smitta, ska vi kunna agera med både omsorg och tydlighet, säger socialminister Jakob Forssmed.

En hälsokontroll kan bland annat innefatta frågor om sjukdomssymtom, temperaturmätning och vid behov provtagning. Hälsokontroller vid inresa kan bli aktuella om det finns misstanke om att en person bär på en samhällsfarlig sjukdom, eller om ett utbrott har skett i ett visst geografiskt område. Beslut om kontroll kan även omfatta andra resenärer som anländer med samma transportmedel. I särskilda fall kan smittskyddsläkaren besluta om karantän, isolering eller ansöka om tvångsundersökning hos domstol.

Vägledningen ska innehålla praktiska råd om hur kontrollerna bör genomföras, samt vilka smittskyddsåtgärder som kan följa. Vägledningen riktar sig främst till smittskyddsläkare, regioner och vårdgivare, och ska utgå från smittskyddslagen. Målet är att stärka beredskapen inför framtida epidemier och pandemier och skapa tydliga rutiner för de aktörer som kan beröras vid ett beslut om hälsokontroll, exempelvis hälso- och sjukvårdspersonal, personal inom gränskontroll och säkerhet och flygplats- och hamnpersonal.

Trots att lagstiftningen möjliggör sådana åtgärder har beslut om hälsokontroll vid inresa aldrig tidigare behövt fattats i Sverige. Regeringen ser därför ett behov av tydlig vägledning för att säkerställa att berörda aktörer vet hur de ska agera om det skulle bli aktuellt.

Uppdraget ska slutredovisas senast den 1 mars 2026.
