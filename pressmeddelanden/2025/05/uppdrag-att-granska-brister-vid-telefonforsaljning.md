# Uppdrag att granska brister vid telefonförsäljning

Konsumentverket ska bedöma om det finns brister i konsumentskyddet vid telefonförsäljning, redovisa vilka åtgärder inklusive regeländringar som myndigheten ser behov av, identifiera branscher eller produktområden där det finns särskilda problem och överväga om det bör göras ytterligare avgränsningar i regleringen, till exempel avseende produkttyper och målgrupper.

− Det är uppenbart att telefonförsäljning behöver granskas ytterligare. Det är helt oacceptabelt att särskilt sårbara grupper såsom äldre, språksvaga och personer med funktionsnedsättningar kommer i kläm vid telefonförsäljning. Vissa produktgrupper och branscher kan vara särskilt problematiska och behöver därför särskilt pekas ut, säger civilminister Erik Slottner.

Via NIX-registret kan konsumenter spärra sitt telefonnummer och sedan 2018 har vi i Sverige ett skriftlighetskrav som syftar till att stärka konsumenternas ställning vid telefonförsäljning. På premiepensionsområdet är telefonförsäljning förbjudet. Men trots dessa åtgärder finns det mycket som tyder på att problemen kvarstår.

Mot denna bakgrund får Konsumentverket i uppdrag att bedöma om det finns brister i konsumentskyddet och redovisa åtgärder för att förbättra skyddet.
