# Regeringen har tagit emot Konkurrensverkets förslag för att öka kvaliteten i offentliga kontrakt

Uppdraget bestod av två delar: att underlätta för upphandlande myndigheter att förkasta onormalt låga anbud och att analysera förutsättningarna för en metod som gör det möjligt att bedöma leverantörers kvalitet. Den första delen redovisades i september. Nu har regeringen tagit emot den andra delen av uppdraget, om ett enhetligt system för leveranskvalitet.

Konkurrensverket föreslår att regeringen går vidare med åtgärder som gör det enklare för upphandlande myndigheter att ta hänsyn till leverantörers tidigare prestationer vid nya upphandlingar. Det skulle öka möjligheterna att utesluta leverantörer som tidigare visat allvarliga brister, även i leveranser till andra organisationer. Det skapar även incitament för leverantörer att upprätthålla hög kvalitet under hela avtalstiden.

– Med ett system för att följa upp kvalitet skapar vi bättre förutsättningar för seriösa aktörer att konkurrera på rättvisa villkor. Det är ett viktigt steg för att höja kvaliteten och värna om skattebetalarnas pengar, säger civilminister Erik Slottner.

I rapporten konstateras att incitamenten att upprätthålla kvalitet i offentliga kontrakt ofta är svaga jämfört med privata affärer. Konkurrensverket föreslår därför bland annat:

* att regeringen utreder ett register över kontrakt som lett till påföljder på grund av allvarliga eller ihållande brister,
* att regeringen uppmuntrar pilotprojekt med avgränsade betygssystem inom områden där risken för oseriösa aktörer är störst, och
* att upphandlande organisationer prioriterar systematisk avtalsuppföljning för att säkerställa att avtalad kvalitet levereras.

Konkurrensverket bedömer att ett nationellt och heltäckande obligatoriskt betygssystem inte bör införas, eftersom nackdelarna överväger fördelarna. Däremot finns förutsättningar för avgränsade lösningar som kan stärka kvaliteten och minska risken för kriminella aktörer. Avgränsade betygssystem kan fungera bra inom områden med många leverantörer och frekventa leveranser, till exempel vissa ramavtal. När flera leveranser betygssätts och upphandlingar sker ofta får betygen större genomslag, särskilt om leverantörer med höga betyg kan ta fler uppdrag.

Ett exempel där Konkurrensverket bedömer avgränsade betygssystem som lämpliga är det nationella ramavtalet om hyrbemanning inom hälso- och sjukvården. Detta ramavtal bygger på gemensamma grundkrav för både läkare och sjuksköterskor, vilket möjliggör rättvisande jämförelser.
