# Socialtjänstministern kallar till sig e-handelsföretag

Efter den uppmärksammade rapporteringen att det förekommit försäljning av sexdockor som föreställer barn via e-handel bjuder socialtjänstminister Camilla Waltersson Grönvall in relevanta aktörer på området till ett möte på Regeringskansliet fredagen den 28 november.

– I ett samhälle där vi gör allt för att skydda barn mot övergrepp kan vi inte acceptera att det säljs sexdockor som föreställer barn. Nu måste vi tillsammans se vilka möjligheter som finns att stoppa förekomsten av dessa, säger socialtjänstminister Camilla Waltersson Grönvall.

Under onsdagen skickas en inbjudan till flera e-handelsföretag, barnrättsorganisationer och myndigheter för att diskutera problematiken, ansvarsområden och potentiella vägar framåt.
