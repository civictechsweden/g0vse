# Regeringen vill göra det lättare att söka till yrkeshögskolan

I dag har varje utbildningsanordnare ett separat system för ansökan. Det gör att en studerande behöver söka utbildningar hos respektive utbildningsanordnare, och att en studerande kan bli antagen till samma yrkeshögskoleutbildning hos flera olika utbildningsanordnare. Bristen på samordning i ansökningsprocessen skapar problem både för den som söker och för de som ska anta sökande till utbildningen. Exempelvis riskerar platser att stå tomma för att studerande antas till utbildningar som de inte påbörjar.

Därför får nu MYH i uppdrag att föreslå hur ett gemensamt system för ansökan och antagning ska utformas. I uppdraget ingår också att undersöka om antagningssystemet för högskolan (NyA) kan användas även för yrkeshögskolan.

– Det måste bli lättare att söka yrkeshögskoleutbildningar, och utbildningsanordnare runt om i Sverige behöver ett effektivare system för att anta studerande. Ett gemensamt ansökningssystem kan bidra till att effektivisera ansökningsprocessen och säkerställa att allt går rätt till, och på sikt stärka den svenska kompetensförsörjningen, säger utbildningsminister Johan Pehrson.
