# Civilministern i Amsterdam för ministermöte om EU:s digitala politik

Vid mötet med D9+ kommer ministrarna samtala om hur användningen av digital teknik i offentlig sektor kan underlättas och hur innovation inom näringslivet kan främjas.

D9+ är ett viktigt sammanhang för att diskutera och driva på den digitala utvecklingen inom EU. Jag uppskattar även att det utgör ett forum för dialog med det europeiska näringslivet, säger civilminister Erik Slottner.

I samband med besöket i Amsterdam reser civilminister Erik Slottner även till Haag för att bland annat träffa Nederländernas digitaliseringsminister Zsolt Szabó samt besöka Sveriges ambassad.
