# Nya åtgärder för att säkra läkemedel

– Det är avgörande att vi har flera olika verktyg för att kunna hantera situationer med brist på läkemedel i Sverige. Genom att ge Läkemedelsverket befogenhet att införa tillfälliga begränsningar får vi ytterligare åtgärder att vidta vid bristsituationer. Med den här ändringen kan vi säkerställa att de mest behövande patienterna får tillgång till nödvändiga läkemedel, även i bristsituationer, säger sjukvårdsminister Acko Ankarberg Johansson.

Läkemedelsbrist kan uppstå av flera olika anledningar, såsom tillverkningsproblem, distributionsproblem, brist på verksamma ämnen eller oväntat hög efterfrågan. För att motverka dessa problem har åtgärder vidtagits både nationellt och internationellt för att stärka hälso- och sjukvården inför brister på läkemedel och för att säkra försörjningsberedskapen.

Utredningen föreslår en lagändring som innebär att Läkemedelsverket ska kunna införa tillfälliga begränsningar vid förordnande och utlämnande av läkemedel, både när en brist uppkommit och när det finns risk för brist. Begränsningarna ska vara nödvändiga för att skydda liv och hälsa och kan utformas på olika sätt, exempelvis genom att begränsa förordnande till vissa yrkesgrupper eller indikationer, eller genom att begränsa mängden läkemedel som får förordnas.

För att underlätta tillämpning och kontroll av föreskrifterna föreslås E-hälsomyndigheten få i uppdrag att utreda möjligheten att införa systemstöd. Inspektionen för vård och omsorg, Läkemedelsverket och länsstyrelserna ska ha tillsynsansvar för att säkerställa att föreskrifterna följs.

Regeringen ser detta som ett viktigt steg för att stärka hälso- och sjukvårdens förmåga att hantera bristsituationer men också för att stärka beredskap. Därigenom vill regeringen säkerställa att alla patienter får den vård och de insatser de behöver.
