# Statsministern talar vid högtidlighållandet av veterandagen

Vid årets ceremoni uppmärksammar Försvarsmakten särskilt de veteraner som tjänstgjorde på Balkan under åren 1991\-2000\. Från regeringen deltar även försvarsminister Pål Jonson och minister för civilt försvar Carl\-Oskar Bohlin.

## Praktisk information för media

Media är välkomna att närvara vid statsministerns tal som hålls klockan 14\.40\. Ackreditering sker genom att kontakta Mikael Wallentin Åström, Presschef Veterandagen, på 073\-852 33 37\.

Talet sänds också på Försvarsmaktens Youtube\-kanal, se länk under genvägar.
