# Möte om säkerheten vid fotbollsmatcher

Vid mötet deltar företrädare för Svenska Fotbollförbundet, Svensk Elitfotboll, Riksidrottsförbundet, Stockholm Live, Spelarföreningen, Svenska Fotbollssupporterunionen samt Polisen och Åklagarmyndigheten. Deltagarna kommer att presentera vilka åtgärder som vidtagits och insatser som gjorts sedan det senaste mötet och vad som planeras inför nästa säsong.

Vid mötet medverkar också regeringens särskilde utredare Anders Hübinette, som i december 2023 fick i uppdrag av regeringen att lämna förslag på åtgärder för att minska våldsamheter och ordningsstörningar vid idrottsarrangemang. Utredningen ska redovisas senast den 31 mars 2025.

Socialminister Jakob Forssmed är tillgänglig för media efter mötet, cirka klockan 12.00. Föranmälan krävs och görs till Johanna Riedel via e-post (se presskontakt) senast tisdag den 5 november klockan 11. Insläpp från cirka klockan 11.30.
