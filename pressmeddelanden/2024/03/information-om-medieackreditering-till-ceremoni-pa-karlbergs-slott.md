# Information om medieackreditering till ceremoni på Karlbergs slott

Efter ceremonin håller statsminister Ulf Kristersson och Socialdemokraternas partiledare Magdalena Andersson tal till Natoländernas ambassadörer i Sverige. Talen hålls på engelska och sker inomhus.

**Tid:** Måndag den 18 mars, kl. 13.00.

**Plats:** Karlbergs slott, ingång från Ekelundsbron. Inpassering för media mellan kl. 11.00-11.30.

**Praktisk information för media:** Det finns möjlighet för media att närvara vid ceremonin och vid talen till ambassadörerna, antalet platser är dock begränsat. Medieackreditering via e-post till Oscar Samuelsson, se presskontakt, senast 13 mars klockan 16.00. Ange namn, personnummer, telefonnummer, redaktion och i vilken funktion du vill delta (reporter eller fotograf). Skicka också med foto på presslegitimation. Journalister som inte har svenskt medborgarskap behöver ange födelsedatum samt passnummer. Observera att du också behöver ange om du vill närvara vid talen till ambassadörerna. Media som får beviljad ackreditering kommer att få bekräftelse och information via e-post. Observera att ceremonin hålls utomhus. Parkeringsplatser finns närmast på Kungsholmen.
