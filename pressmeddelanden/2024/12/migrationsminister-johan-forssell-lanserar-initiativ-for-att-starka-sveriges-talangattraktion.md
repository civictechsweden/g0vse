# Migrationsminister Johan Forssell lanserar initiativ för att stärka Sveriges talangattraktion

Syftet med rådet är att få medskick om vad som kan göras för att främja Sveriges talangattraktion och locka internationell spetskompetens.

– Vi ser hur andra länder aktivt arbetar för att positionera sig i den globala konkurrensen om talanger. Jag vill att Sverige ska vara nummer ett i EU när det kommer till talangattraktion. Genom att samla expertis från näringslivet kan vi ta viktiga steg för att säkerställa att Sverige inte bara behåller utan också stärker sin position som ett globalt kunskapsnav, säger migrationsminister Johan Forssell.

Näringslivsrådet diskuterade bland annat förenklade migrationsprocesser, strategiska samarbeten och prioriterade geografiska områden.

– Näringslivsrådet kommer att bidra med värdefulla insikter och rekommendationer kring hur Sverige kan förbättra sina strategier för att attrahera och behålla talanger inom kritiska områden som IT, teknik, forskning och medicin, säger migrationsministern.

Näringslivsrådet kommer att bestå av:

* Katarina Berg, Spotify
* Jessica Häggström, Investor
* Fredrik Hillelson, Novare
* Daniella Waldfogel, Stockholms Handelskammare
* Ellinor Johansson, Embracer
* MajBritt Arfert, Ericsson
* Masih Yazdi, SEB

Näringslivsrådet kommer att hålla ytterligare möten under våren.
