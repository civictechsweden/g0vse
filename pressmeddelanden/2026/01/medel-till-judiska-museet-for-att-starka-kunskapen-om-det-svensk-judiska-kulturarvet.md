# Medel till Judiska museet för att stärka kunskapen om det svensk-judiska kulturarvet

– Att stärka kunskapen om det svensk‑judiska kulturarvet skapar förutsättningar för att främja judiskt liv i Sverige. Genom att utveckla arbetet med den judiska minoritetens kulturarv på museer i hela landet bidrar Judiska museet till att genomföra regeringens strategi för att stärka judiskt liv och motverka antisemitism, säger kulturminister Parisa Liljestrand.

Regeringen beviljade Stiftelsen Judiska museet 275 000 kronor för kunskapshöjande insatser kring det svensk-judiska kulturarvet under 2025. För 2027 avser regeringen att avsätta ytterligare 1,5 miljoner kronor för ändamålet. Syftet med insatserna är att stärka arbetet med att bevara och synliggöra det svensk-judiska kulturarvet.
