# Johan Forssells besök i Uppsala

Vid besöket på Åklagarkammaren diskuterades hur skärpta regler för utvisning på grund av brott kan komma att påverka kammarens arbete. På Uppsala universitet deltog ministern i ett möte om ett nytt lagförslag som bland annat innehåller förslag på förbättringar för forskare och doktorander som vill verka i Sverige.
