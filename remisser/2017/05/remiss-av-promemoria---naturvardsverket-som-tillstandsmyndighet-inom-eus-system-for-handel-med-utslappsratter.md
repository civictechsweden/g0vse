# Remiss av Promemoria - Naturvårdsverket som tillståndsmyndighet inom EUs system för handel med utsläppsrätter

Här kan du ta del av de remissvar som har kommit till Miljö\- och energidepartementet gällande remissen Promemoria \- Naturvårdsverket som tillståndsmyndighet inom EU:s system för handel med utsläppsrätter.


Remissvaren ska ha kommit in till Miljö\- och energidepartementet senast den 4 augusti 2017\.
