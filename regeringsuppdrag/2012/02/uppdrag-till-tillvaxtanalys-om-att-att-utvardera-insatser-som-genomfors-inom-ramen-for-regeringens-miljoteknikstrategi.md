# Uppdrag till Tillväxtanalys om att att utvärdera insatser som genomförs inom ramen för regeringens miljöteknikstrategi

Regeringen uppdrar åt Myndigheten för tillväxtpolitiska utvärderingar och analyser (Tillväxtanalys) att utvärdera de insatser som genomförs inom ramen för den av regeringen antagna strategin för utveckling och export av miljöteknik 2011–2014, den så kallade miljöteknikstrategin.
