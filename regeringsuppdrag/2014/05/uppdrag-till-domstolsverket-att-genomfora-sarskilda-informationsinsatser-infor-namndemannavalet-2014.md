# Uppdrag till Domstolsverket att genomföra särskilda informationsinsatser inför nämndemannavalet 2014

Regeringen uppdrar åt Domstolsverket att inför nämndemannavalet år 2014 genomföra särskilda informationsinsatser gentemot såväl de nominerande och väljande organen som allmänheten. Syftet är att få till stånd en bredare rekrytering av nämndemän och en ökad kunskap hos presumtiva nämndemän om vad nämndemannauppdraget innebär.
