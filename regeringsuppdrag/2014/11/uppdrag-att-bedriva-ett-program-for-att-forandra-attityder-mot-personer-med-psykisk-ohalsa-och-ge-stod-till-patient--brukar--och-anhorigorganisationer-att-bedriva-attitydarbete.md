# Uppdrag att bedriva ett program för att förändra attityder mot personer med psykisk ohälsa och ge stöd till patient-, brukar- och anhörigorganisationer att bedriva attitydarbete

Regeringen beslutar att ge Myndigheten för delaktighet (MFD) i uppdrag att i nära samverkan med Nationell Samverkan för Psykisk Hälsa (NSPH) bedriva ett riksomfattande program för att öka kunskapen om och förändra attityder till personer med psykisk ohälsa. Uppdraget ska redovisas i sin helhet till Socialdepartementet senast den 15 juli 2015.
