# Ändring av beslut om uppdrag att genomföra regionala dialoger för utbyggnaden av mobilnät

Då de regionala dialogerna skulle kunna ge värdefulla bidrag till genomförandet av uppdraget till Post\- och telestyrelsen (PTS) att se över förutsättningarna för samhällsmaster bör datumen för rapportering av uppdraget att genomföra regionala dialoger senareläggas för att säkerställa att viss samordning mellan uppdragen kan ske.
