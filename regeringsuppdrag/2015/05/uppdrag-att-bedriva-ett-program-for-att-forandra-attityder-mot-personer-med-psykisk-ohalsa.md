# Uppdrag att bedriva ett program för att förändra attityder mot personer med psykisk ohälsa

Regeringen beslutar att ge Myndigheten för delaktighet (MFD) ett uppdrag att i nära samverkan med Nationell Samverkan för Psykisk Hälsa (NSPH) bedriva ett riksomfattande program för att öka kunskapen om och förändra attityder till personer med psykisk ohälsa under 2015\.

Uppdraget ska redovisas till Socialdepartementet senast den 29 februari 2016\.
