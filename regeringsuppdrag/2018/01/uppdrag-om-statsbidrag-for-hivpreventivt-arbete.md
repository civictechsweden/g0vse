# Uppdrag om statsbidrag för hivpreventivt arbete

Regeringen uppdrar åt Folkhälsomyndigheten att förbereda fördelningen av statsbidrag till ideella organisationer som bedriver hivpreventivt arbete på regional och lokal nivå för 2018\.


Myndigheten ska vid förberedelserna för fördelnignen av bidraget särskilt sträva efter ett tillvägagångssätt som medför att organisationerna tidigt kan erhålla medel avseende 2018 års insatser. Vidare ska utgångspunkten vara att bidraget ska fördelas direkt till de ideella organisationerna.

Vid genomförande av uppdraget ska myndigheten sträva efter att de insatser som finansieras via anslaget stödjer det arbete som bedrivs av landsting och kommuner avseende hiv/aids och andra smittsamma sjukdomar.
